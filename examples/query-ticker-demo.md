# Example: Ticker view — $SIVE

User prompt:

```text
用 serenity-twin：Serenity 对 $SIVE 的看法是什么？包括 conviction tier、演变和主要风险。
```

Expected agent behavior:

1. Read `corpus/references/theses.md` → SIVE entry under Optical / CPO / photonics
2. Optional: `python scripts/radar.py --json` → confirm SIVE in Conviction watch / Heating
3. Output structure (中文):

---

**她的观点**

Serenity 将 SIVE (Sivers Photonics) 定位为 CPO/硅光链路中的 **merchant laser 卡点**——多家 leading CPO 玩家（Ayar、Lightmatter 等）的 primary/sole source 路径经过 Sivers；Win Semi 是扩产瓶颈，但 Sivers 在产能分配紧张时本身也成为 bottleneck 的一环。

**Conviction tier 演变**

- 2025 末–2026 初：高 conviction core，类比「LITE 早期的样子」
- 2026 春：OFC/CEO 表态 sold-out、substrate 焦虑 → 强化上游 AXTI 逻辑，SIVE 仍为 photonics 组合核心

**最新 stance（corpus 截至 ~2026-06）**

Structural long；相对 LITE 仍被视为新资金更优先的 pure-play laser entry。

**主要风险**

- 产能/partner（Win Semi）能否按时爬坡
- 多 source 化进度
- 小盘波动、稀释风险
- Thesis 会 decay — 需核对最新财报与客户认证

**什么情况说明判断错了**

- 主要 CPO 客户完成 second source 且 Sivers 份额下滑
- 产能承诺反复跳票且无订单/收入兑现
- 大幅 ATM/稀释 without capacity payoff

仅作信息跟踪，不构成投资建议。
